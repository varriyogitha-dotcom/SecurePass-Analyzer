# SecurePass-Analyzer
## Description
SecurePass Analyzer is a Python project that checks the strength of a password based on:
- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

It provides a security score, password strength, and suggestions to improve weak passwords.

## Features
- Password Strength Analysis
- Security Score (0–10)
- Strength Meter
- Improvement Suggestions

## Technologies Used
- Python
- Regular Expressions (re)

## How to Run
1. Install Python.
2. Download the project.
3. Open a terminal in the project folder.
4. Run:
   python securepass.py
